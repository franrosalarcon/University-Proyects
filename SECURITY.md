# Security Policy
This repository hosts university projects. Security updates are not relevant, but the code is preserved for learning and practice.
## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 5.1.x   | :white_check_mark: |
| 5.0.x   | :x:                |
| 4.0.x   | :white_check_mark: |
| < 4.0   | :x:                |

## Reporting a Vulnerability

If you notice any issues with the code, you can report them by opening an issue in this repository.

Contact the project maintainer via franrosalarcon@gmail.com
Include a description of the issue and steps to reproduce it.
You will receive updates about your report.
Accepted issues will be fixed promptly; declined reports will be explained.
